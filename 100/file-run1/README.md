# file-run1

### Description

A program has been provided to you, what happens if you try to run it on the command line?

### Resources

run (program)

### Solution

Step 1: Use chmod to change the access permissions for the file

```console
sam@ubuntu:~$ chmod +x run
```

Step 2: Run the run file

```console
sam@ubuntu:~$ ./run
```
